# Pixelfy Launcher
 

Magisk Module to get the Android 11 Launcher on unnsupported devices/roms. 

* This is a beta. Most things work, but if you find any issues, please report them under the "issues" tab

## Requirements:

* Magisk 18 or higher
* Android 11 only
